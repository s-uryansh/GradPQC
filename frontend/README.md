# GradPQC Web Dashboard

Frontend interface for the GradPQC Quantum Migration Intelligence Platform. Built with Next.js, it visualizes the Cryptographic Bill of Materials (CBOM) generated by the Go scanner.

## Prerequisites
- Node.js (v18+)
- pnpm
- MySQL Server

## Installation & Setup

**1. Install Dependencies**
```bash
cd apps/web
pnpm install
```

**2. Database Setup**
Execute the following in your MySQL server to configure authentication:
```sql
CREATE DATABASE gradpqc;
USE gradpqc;

CREATE TABLE user (
  id VARCHAR(36) PRIMARY KEY,
  email VARCHAR(255) UNIQUE NOT NULL,
  passwordHash VARCHAR(255) NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**3. Environment Variables**
Create a `.env.local` file inside the `apps/web` directory:
```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_NAME=gradpqc
JWT_SECRET=your_secure_random_jwt_secret_key_here
```

## Usage

**1. Start the Development Server**
```bash
pnpm run dev
```

**2. Access the Application**
- Open `http://localhost:3000` (or `http://localhost:3001` depending on terminal output).
- Click **Create account** on the login screen to register a new user.
- Sign in to view the PQC Posture, Cyber Rating, and Asset Discovery dashboards.
